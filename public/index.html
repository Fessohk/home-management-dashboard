What is gertieHQ?

A home management dashboard tracking energy use and running costs. Built for an Edwardian house in West Bridgford, Nottingham.

Data sources

Electricity pulled live from the Octopus Energy API using your MPAN and smart meter serial. Half-hourly data throughout. Gas paused due to a smart meter communication issue.

How it works

Netlify serverless functions call Octopus on each page load. API keys never touch the browser. Tags and memory stored in Netlify Blob storage. AI analysis via Anthropic API (Claude Sonnet).

Data lag

Octopus uploads half-hourly data with a 1-2 hour delay. Days with fewer than 20 slots are treated as incomplete.

Tariff

Octopus Flexible (VAR-22-11-01). Rate: 24.99p/kWh electricity.
