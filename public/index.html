What is gertieHQ?

A home management dashboard tracking energy use, running costs, and everything in between. Built for an Edwardian house in West Bridgford, Nottingham.

Data sources

Electricity pulled live from the Octopus Energy API using your MPAN and smart meter serial. Half-hourly data throughout. Gas paused due to a smart meter communication issue. Weather integration planned.

How it works

A Netlify serverless function calls Octopus on each page load. Your API key never touches the browser. Tags and memory stored in Netlify Blob storage. AI analysis via Anthropic API.

Data lag

Octopus uploads half-hourly data with a 1-2 hour delay. Days with data only before 10:00 are treated as incomplete and the previous day is shown instead.

Tariff

Octopus Flexible (VAR-22-11-01). Rate: 24.99p/kWh electricity.
